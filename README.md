## 🧠 About This Project

This is not a traditional portfolio.

Instead of scrolling through pages, you explore a fully interactive 3D world built in Unity.

You control a tank, navigate the environment, interact with different zones, and experience projects through gameplay.

The goal was simple:  
**Turn a static portfolio into something interactive, engaging, and memorable.**

---

## 🎮 What You Do

- Spawn as a tank in a 3D environment  
- Explore different zones representing projects  
- Interact with pads to open links and content  
- Complete optional tasks to understand mechanics  
- Destroy objects and experiment with physics  

---

## 🧩 Key Features

- 💥 **Destruction System**  
  Real-time destruction of objects with physics-based feedback  

- 🎯 **Projectile System + Object Pooling**  
  Efficient shell system using pooling for performance  

- 🧠 **Interaction System**  
  Context-based interaction with environment and UI  

- 🗺️ **Exploration & Navigation System**  
  Guided tasks to help users discover features naturally  

- 📦 **Scriptable Object Driven Content**  
  Modular and scalable data handling for portfolio items  

- 🌿 **Procedural Environment Elements**  
  Random spawning of trees and grass with frustum culling for optimization  

---

## 🕹️ Controls

### Movement
- **W / S** → Move Forward / Reverse  
- **A / D** → Turn Left / Right  
- **SHIFT** → Boost  
- **E** → Interact / Use Pad  
- **R** → Reset (if stuck)  

### Combat
- **Left Click** → Shoot  
- **Right Click** → Aim  
- **Mouse** → Look Around (while aiming)  
- **TAB** → Tank Panel  

### Menu
- **ESC / M** → Open / Close Menu  

---

## 🎯 Tasks System

The experience includes optional objectives that guide the player:

- Explore all zones  
- Destroy objects  
- Travel distance  
- Reset vehicle  
- Eliminate turrets  

These are not mandatory but help users understand core mechanics.
